# AAT Backend

## Setup

1. Create a `.env` file (you can copy from `.env.example`).
2. Install dependencies:

```bash
npm install
```

## Run

```bash
npm run dev
```

The API will start on `http://localhost:5000` by default.

## Routes

- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/users`
- `PUT /api/users/:id`
- `DELETE /api/users/:id`
- `POST /api/records`
- `GET /api/records`
- `GET /api/records/:id`
- `PUT /api/records/:id`
- `DELETE /api/records/:id`
